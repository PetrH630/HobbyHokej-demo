# UML diagramy service modulů

PlantUML diagramy pro moduly:
- `playerservices`
- `user.services`
- `registration.services`
- `notification.services`
- `season.services`
- `match.services`

Každý modul má vlastní podsložku s README a sadou strukturálních i sekvenčních diagramů.
Diagramy jsou připravené tak, aby ukazovaly architekturu, hlavní vazby mezi service třídami a nejdůležitější integrační body na repository, mappery a ostatní moduly.
