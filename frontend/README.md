
# HobbyHokej – Fullstack Application Documentation

Tento projekt je webová aplikace určená pro správu hobby hokejových zápasů.
Aplikace umožňuje hráčům registraci na zápasy, správu sestav a sledování statistik.

## Technologie

Frontend
- React
- Vite
- Bootstrap
- Axios

Backend
- Spring Boot
- JPA / Hibernate
- MySQL
- REST API

## Hlavní funkce systému

- registrace hráčů na zápasy
- správa zápasů administrátorem
- správa hráčů a uživatelů
- notifikační systém
- statistiky hráčů
- správa sezón
