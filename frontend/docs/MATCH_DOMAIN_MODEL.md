
# Match Domain Model

Základní entitou systému je **Match**.

Match obsahuje:

- datum a čas zápasu
- místo konání
- maximální počet hráčů
- režim zápasu
- výsledek zápasu
- skóre

Další související entity:

Player
- reprezentuje hráče systému

MatchRegistration
- registrace hráče na zápas

Team
- tým hráče v zápase

Season
- sezóna zápasů

MatchScore
- value object reprezentující skóre
